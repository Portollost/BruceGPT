# BruceGPT
## Instalação

 1. Baixe o [NodeJS](https://nodejs.org/en/download) >= 18.15.0
 2. Siga o passo a passo para instalar o flowise:

	    npm install -g flowise
	    npx flowise start
	    npx flowise start
 3. Abra o [http://localhost:3000](http://localhost:3000/)
